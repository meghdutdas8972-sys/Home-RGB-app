🌈 Home RGB app (ESP8266 + WS2812B)

A simple and powerful IoT app to control WS2812B RGB LEDs using an ESP8266 over WiFi.
This project allows you to change colors, brightness, and effects of your LED strip using a mobile-friendly interface.

---

📱 Features

- 🎨 Control RGB colors in real-time
- 💡 Adjust brightness easily
- 🌈 Multiple lighting effects (fade, rainbow, etc.)
- 📶 WiFi-based control (no Bluetooth required)
- 📲 Works on any phone browser
- ⚡ Fast and responsive interface

---

🔧 Hardware Required

- ESP8266/ESP32
- WS2812B LED strip
- Power supply (5V recommended)
- Connecting wires

---

💻 Software Used

- Arduino IDE
- ESP8266 WiFi Library
- FastLED / NeoPixel Library


🚀 Getting Started

1. Install ESP8266 board in Arduino IDE
2. Install required libraries
3. Upload the code to ESP8266
4. Connect to WiFi (check Serial Monitor for IP address)
5. Open the IP address in your browser
6. Start controlling your RGB LEDs 🎉
